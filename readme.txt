== Installation ==

1. Unzip just-widget-link.zip
2. Upload the `just-widget-link` directory and all its contents into your `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use the widget admin page to add the widget to one or more of your sidebars and configure it
